# astrbot_plugin_kami — 卡密发放

群内卡密发放插件。用户在群内发送指令即可领取卡密，机器人私发卡密内容。支持群白名单、领取冷却、WebUI 可视化管理卡密池。

## ✨ 功能

- **群内领取**：在群聊中发送 `/getkami`（指令名可在后台自定义），机器人私发一张卡密
- **群白名单**：只有白名单内的群才能使用卡密功能
- **领取冷却**：同一用户领取后需等待指定时间才能再次领取（可在 WebUI 配置）
- **管理员重置**：管理员可通过 `/resetkami` 指令或 WebUI 重置用户领取状态
- **一键重置**：一键清除所有已领取的旧卡密，未使用的卡密保留
- **批量导入**：支持文本框粘贴或从 `.txt`/`.csv` 文件批量导入卡密，自动去重
- **WebUI 管理**：在 AstrBot 后台可视化管理卡密池、查看领取记录、配置白名单与冷却时间

## 📦 安装

1. 将插件文件夹放入 AstrBot 的 `data/plugins/` 目录
2. 在 AstrBot WebUI → 插件管理 中启用「卡密发放」
3. 进入插件详情页，打开管理面板添加卡密并配置群白名单

## 🎮 指令

| 指令 | 别名 | 说明 | 权限 |
|------|------|------|------|
| `/getkami`（可自定义） | — | 群内领取卡密，机器人私发。**指令名可在 WebUI 配置面板修改** | 群成员 |
| `/resetkami` | `重置卡密` `重置领取` | 重置用户领取状态，支持 @某人 | 管理员 |
| `/kami_status` | `卡密状态` | 查看卡密池统计信息 | 管理员 |

> 💡 领取指令默认是 `/getkami`，你可以在 WebUI「配置设置」面板中修改为任意自定义指令（如 `/领取`、`/claim` 等）。

## ⚙️ 配置

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `claim_command` | string | `getkami` | 领取卡密指令（不含 `/`），可在 WebUI 修改 |
| `cooldown_hours` | int | 24 | 领取冷却时间（小时），设为 0 无冷却 |
| `whitelist_groups` | list | [] | 群白名单，留空不限制 |

## 🖥️ WebUI 管理页面

插件提供三个管理面板：

- **卡密管理**：批量添加卡密（文本粘贴或从文件导入），查看可用/已领取状态，删除卡密，一键重置清除旧卡密
- **领取记录**：查看所有领取明细，按用户重置状态
- **配置设置**：自定义领取指令、修改冷却时间和群白名单
